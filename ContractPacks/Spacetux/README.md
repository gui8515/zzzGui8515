# Spacetux Contract Packs

A collection of Contract Packs for Kerbal Space Program by Spacetux.

## Description

This collection includes three distinct contract packs:

### Grand Tours
Challenging contracts for grand tour missions visiting multiple planets and moons in a single mission.

### Rover Missions
Contracts focused on deploying and operating rovers on various celestial bodies.

### Unmanned Contracts
Contracts specifically designed for unmanned/robotic missions and probes.

## Installation

1. Install [Contract Configurator](https://forum.kerbalspaceprogram.com/index.php?/topic/91625-contract-configurator/) (required dependency)
2. Extract this folder to your KSP `GameData/ContractPacks/` directory

## Dependencies

- Contract Configurator
